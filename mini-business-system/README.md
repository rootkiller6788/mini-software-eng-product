# mini-business-system — 业务系统 (C 语言实现)

## 概述

轻量级 C99 业务系统库，实现领域驱动设计(DDD)、工作流引擎、规则引擎、状态机、审批流程五大模块。

## 模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 领域模型 | `domain_model.h/.c` | 限界上下文、实体、值对象、聚合根、领域事件、统一语言术语表、上下文映射 |
| 工作流引擎 | `workflow_engine.h/.c` | BPMN 式流程定义、流程实例、任务分配、并行/排他网关、定时边界事件 |
| 规则引擎 | `rule_engine.h/.c` | 产生式规则、优先级/显著性、事实插入→正向链推理(Rete网络)、决策表、规则版本、审计追踪 |
| 状态机 | `state_machine.h/.c` | 状态(进入/退出/执行动作)、转换(事件→守卫→动作)、层次状态、历史状态、并行状态、事件延迟 |
| 审批流程 | `approval_process.h/.c` | 提交→审核→批准/驳回→升级、多级审批(串行/并行/动态)、条件审批、委托、超时自动升级、审计日志 |

## 构建

```sh
make
```

